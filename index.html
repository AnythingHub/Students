    Settings

    Bring your own keys for Google Drive, Google Search, and optional cloud sync. Everything works locally without these.

    Google (Drive & Search)

    Create one at console.cloud.google.com → APIs & Services → Credentials. Enable the "Google Drive API" and "Custom Search API" for it.
    Create a search engine at programmablesearchengine.google.com set to search the entire web, then copy its Search Engine ID.

    Supabase sync (optional)

    Optional — this app already saves everything locally to your account automatically. Only fill this in if you want a second copy in your own Supabase project (e.g. to access it from another tool). Create a table first:
    create table study_data (id text primary key, payload jsonb, updated_at timestamptz default now());
